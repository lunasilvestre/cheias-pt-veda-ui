    ## About cheias.pt

    cheias.pt is a flood monitoring platform for Portugal, combining satellite imagery, hydrological data, and climate science to document and analyze flood events. The platform integrates real-time data from multiple sources to support emergency response, climate adaptation planning, and public awareness.

    ### Data Sources

    The platform draws on freely available Earth observation and meteorological data including Copernicus Sentinel-1 SAR flood mapping, Sentinel-2 optical imagery, ERA5 reanalysis (precipitation, soil moisture, wind), NOAA sea surface temperature, GloFAS river discharge forecasts, and the Portuguese national hydrological monitoring network (SNIRH).

    ### Technology

    cheias.pt is built on the VEDA Dashboard framework developed by NASA IMPACT / Development Seed. Raster data is served as Cloud-Optimized GeoTIFFs (COGs) through a TiTiler instance. Vector flood extents use PMTiles for efficient web delivery.

    This project is open source and built on openly maintained projects including React, MapLibre GL, Parcel, STAC, and TiTiler.
